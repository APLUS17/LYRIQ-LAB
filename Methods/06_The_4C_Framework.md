## Core Technique

> *To be populated from source PDF.*

**Baseline logic (until PDF is parsed):**

The 4C Framework is the Architect's primary tool. It is applied sequentially — you cannot skip steps.

```
COPY   → Find the strongest existing line. This is your north star.
CUT    → Remove everything that competes with or dilutes the north star line.
CARVE  → Sharpen what remains: tighter syllables, harder words, cleaner images.
CONNECT → Ensure surviving lines form a single emotional arc from first to last.
```

The 4C pass transforms a Lyricist raw draft into a structured, scoreable output.

## Prompt Template

> *To be populated from source PDF.*

**Baseline prompt:**
```
4C FRAMEWORK — Architect Pass

Input: [raw Lyricist draft]

STEP 1 — COPY
Read the draft once. Circle the one line you would never cut.
That line is your north star. Write it here: _______

STEP 2 — CUT
Go line by line. Ask: "Does this line make the north star line stronger?"
If not, cut it. Be ruthless. Cut at least 30% of the draft.

STEP 3 — CARVE
For each surviving line:
- Replace any soft word (nice, beautiful, very, feel, think) with a concrete alternative
- Recount syllables. Target: verse 8–10 / pre 6–8 / chorus 6–9
- Eliminate filler words: just, really, kind of, so, sort of

STEP 4 — CONNECT
Read the draft aloud. Map the emotional journey:
Line 1 → [state]. Middle → [shift]. Final line → [resolution or open wound].
If the arc isn't clear, reorder or bridge until it is.
```

## Scoring Anchor

The 4C Framework affects all four scoring dimensions but is the primary driver of **Structural coherence (20%)**.
A draft that has been through a full 4C pass will show measurably higher flow and structural scores than the raw draft.
The Critic compares draft_a (raw) against draft_b (post-4C) — the delta reveals how well the Framework was applied.
